Diagrama de Dispersion y sus Tipos

Diagrama de Dispersión

Un diagrama de dispersión es una herramienta gráfica que muestra la relación entre dos variables cuantitativas. Se utiliza para analizar si existe una correlación o dependencia entre las variables, y qué tipo y grado de correlación hay. Para hacer un diagrama de dispersión, se representan los valores de las dos variables como puntos en un plano cartesiano, y se observa la forma y dirección de la nube de puntos resultante.

Se emplea cuando una o varias variables está bajo el control del experimentador. Si existe un parámetro que se incrementa o disminuye de forma sistemática por el experimentador, se le denomina parámetro de control o variable independiente y habitualmente se representa a lo largo del eje horizontal (eje de las abscisas). La variable medida o dependiente usualmente se representa a lo largo del eje vertical (eje de las ordenadas). Si no existe una variable dependiente, cualquier variable se puede representar en cada eje y el diagrama de dispersión mostrará el grado de correlación (no causalidad) entre las dos variables.

Tipos

Los tipos de correlación que se pueden identificar en un diagrama de dispersión son los siguientes:

Correlación positiva: 

Se presenta cuando ambas variables aumentan o disminuyen juntas, es decir, hay una relación directa o proporcional entre ellas. La nube de puntos tiene una forma alargada y una pendiente positiva, como se muestra en el siguiente ejemplo:

Correlación nula:

Se presenta cuando no hay una relación aparente entre las variables, es decir, son independientes o no se afectan entre sí. La nube de puntos tiene una forma circular o elíptica y no tiene una dirección definida, como se muestra en el siguiente ejemplo:

El grado de correlación se refiere a qué tan fuerte o débil es la relación entre las variables, y se puede medir mediante el coeficiente de correlación, que es un número entre -1 y 1. Cuanto más cercano sea el valor absoluto del coeficiente a 1, más fuerte será la correlación, y cuanto más cercano sea a 0, más débil será. Por ejemplo, un coeficiente de 0.9 indica una correlación positiva muy fuerte, mientras que un coeficiente de -0.2 indica una correlación negativa muy débil.
